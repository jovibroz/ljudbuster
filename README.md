# LjudBuster

Intern FastAPI-baserad nedladdare/resolver för ljud och video.

## Struktur

- `app/` - applikationskod
- `app/templates/` - frontend/Jinja-mallar
- `state/` - lokal/live state, ignoreras av git
- `logs/` - lokala loggar, ignoreras av git
- `scripts/` - hjälpskript
- `config/` - konfigurationsexempel
- `archive/` - manuella arkiv/backuper
